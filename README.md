# Greenhouse Control

Bu proje, bir serayı kontrol etmek için çeşitli sistemlerin yönetimini sağlar.
Qt VE pyQt5 kullanıldı.

## Sistemler

- Sıcaklık Kontrolü
- Aydınlatma Sistemi
- Nem Kontrolü
- Sulama Sistemi
- Toprak Asitliği İzleme
